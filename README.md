## Basic Shapes

Example template:
```
<svg width="200" height="200">
</svg>
```

### Rectangle

Start off simple: x, y, width and height. rx and ry set round corners.

```
<svg width="200" height="200">
    <rect x="10" y="10" width="30" height="30"/>
    <rect x="60" y="10" width="30" height="30" rx="10" ry="10"/>
</svg>
```
